LLM CV PDF Parser

GDPR-safe mode: uploaded CVs are used only for parsing and deleted immediately after parsing. No raw CV PDF URL or unredacted text is stored.

You can select one or many CV PDFs.
Mistral Direct API: mistral-small-latest Gemma: google/gemma-2-9b-it via OpenRouter Llama: meta-llama/llama-3.1-8b-instruct via OpenRouter
One click runs all three parsers and saves one result row per file per model.
